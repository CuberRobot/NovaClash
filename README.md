# NovaClash

> **当前版本：v1.4**（Bug 修复稳定版）

这是一个回合制策略桌游模拟器，用于测试角色平衡性和游戏机制。两名玩家各获得一个角色池，每回合选择角色并应用增益，进行三局两胜制的对战。

## 快速开始

### 命令行模式
```bash
python main.py
```

### Web 模式
```bash
pip install -r requirements.txt
uvicorn webserver:app --host 0.0.0.0 --port 8000
```

## 游戏模式

### 标准模式（Standard）
- **角色池**：6 选 3
- **增益次数**：4 次
- **战斗顺序**：A1→B1→A2→B2→A3→B3

### 混沌模式（Chaos）
- **角色池**：6 选 3（排除自爆步兵）
- **增益次数**：4 次
- **特色**：标签随机分配（每个玩家 4 个标签，每角色 0-2 个）
- **互斥规则**：自爆 + 复活/吸血/狂暴，狂暴 + 重装

### 大战场模式（Big Battlefield）
- **角色池**：9 选 5
- **增益次数**：6 次
- **战斗顺序**：A1→B1→A2→B2→A3→B3→A4→B4→A5→B5

**通用规则**：
- 三局两胜制
- 每回合选择角色后应用增益（ATK+2 或 HP+4）
- 攻击目标选择：优先 HP 最低或 ATK 最高的敌人

## 角色与标签

### 12 种标签效果

| 序号 | 标签 | 描述 |
| :--- | ---- | ---- |
| 1 | 自爆 | 攻击时自身必死；目标 max HP≤26 直接击杀；自爆伤害不参与护盾 |
| 2 | 群体治疗 | 己方角色被击杀时消耗次数以 30% 最大血量复活（自爆步兵不可被复活） |
| 3 | 标签剥夺 | 战斗开始时剥夺对方一名有标签角色的全部标签 |
| 4 | 护盾 | 为队友分担 50% 伤害至自身，有次数限制 |
| 5 | 重装盔甲 | 受到≥6 点伤害时按 60% 结算 |
| 6 | 箭矢穿透 | 若攻击目标为脆皮 (max HP≤24)，命中后对另一名敌人追加减半伤害 |
| 7 | 狂暴 | HP≤14 时伤害×1.4 |
| 8 | 中毒 | 叠加型 DOT，每轮次造成伤害，可叠加但有上限 |
| 9 | 群伤 | 对对方所有存活单位造成等量伤害 |
| 10 | 吸血 | 造成伤害的 25% 转为回复，单次上限 5；自爆不触发 |
| 11 | 反弹 | 实际受到伤害的 40% 反弹给攻击者，反弹伤害不触发护盾/反弹/重装 |
| 12 | 斩杀 | 目标当前 HP ≤ max HP 25% 时，伤害×1.5 |
