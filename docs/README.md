## 版本发布流程

每次发布新版本（如 v1.4）时：

1. 将 `设定.md` 复制为 `archive/v1.3-设定.md`
2. 将 CLAUDE.md 中已完成内容整理为 `archive/v1.4-更新日志.md`
3. 更新 `设定.md`（添加新角色/新标签设定）
4. 更新 CLAUDE.md（新的待开发计划）
5. 更新根目录 `README.md` 的版本导航表
