# NovaClash v1.0 更新日志

**发布日期**：2026-02

## 核心功能

### 核心玩法
- 两人三局两胜制
- 每局从随机 6 人池选 3 人组队
- 4 次增益机会（ATK+2 / HP+4）
- 选定攻击策略后自动回合制战斗
- 先手权按双方「总攻击 + 总血量」之和较小一方获得

### 角色与标签
- **12 名角色**实装
- **9 种标签**全部实现：自爆、群体治疗、标签剥夺、护盾、重装盔甲、箭矢穿透、狂暴、中毒、群伤
- 战斗以事件标记列表输出

### 双端支持
- **命令行版** (`main.py`)：本地双人同屏测试
- **Web 版** (`webserver.py`)：FastAPI + WebSocket
  - 支持随机匹配
  - 房间号创建/加入
  - 两人到齐后自动开始对局

### 前端功能
- 单页应用设计
- 匹配区、角色池与出击位
- 增益与策略配置
- 战报与原始事件展示
- 历史对局（最近 8 场本地存储）

### 安全与维护
- 历史对局展示对昵称与战报做 HTML 转义防 XSS
- 中英文注释补充

### 对拍测试
- `tests/test_battle.py`：随机选角与策略
- 多进程跑多场对局并导出紧凑日志
- 供平衡性分析使用

## 技术栈

- **后端**：Python + FastAPI + WebSocket
- **前端**：原生 HTML/CSS/JS
- **运行方式**：`uvicorn webserver:app` 或 `python main.py`
