# 星陨（NovaClash 设定集）

本文档为 NovaClash 的世界观与角色设定，与游戏内规则、角色名、标签名一致。

## 世界观设定：星辰竞技场
在无尽的宇宙深处，存在着一个由古老星能水晶构筑的次元——星陨界。这里没有恒星，也没有行星，只有无数漂浮的几何水晶碎片，它们反射着遥远星系的微光，构成了梦幻般的低多边形宇宙。星云如轻纱般缠绕着巨大的水晶结构，光线在棱角间折射，形成永不熄灭的、冷艳的荧光。

传说，星陨界是宇宙诞生时第一缕意识的梦境。在那场创世大爆炸中，散落的能量凝聚成了十二枚本源水晶，每一枚都承载着一种战斗的本能——狂暴、守护、诅咒、重生……它们被放置在不同的星域，逐渐孕育出了各具特色的星灵战士。这些战士并非血肉之躯，而是由水晶的几何切面构成，他们的身体会随着情绪和战斗状态改变光芒：愤怒时转为炽烈的橙红，冷静时化为幽蓝，濒死时则如碎裂的星辰般四散。

每隔千年，星陨界会迎来一次 “归寂潮”——宇宙暗物质周期性涌来，试图吞噬所有光芒。为了抵抗暗潮，十二枚本源水晶会召唤各自的星灵战士，在星核竞技场中进行一场又一场的模拟战斗。这并非杀戮，而是通过战斗激发水晶的共鸣，凝聚足够的光能来驱散黑暗。每一次攻击、每一次增益，都会让水晶的光芒更盛一分。

玩家扮演的是一位 “星痕调停者” ，被本源水晶选中，负责在归寂潮期间调配战士、制定策略。你无法直接干预战斗，但可以通过选择不同的战士组合、分配增益（即水晶碎片能量），来引导战局。对手则是另一位调停者，你们在璀璨的星云棋盘上展开三局两胜的博弈。胜利的一方将为自己的水晶汲取更多星能，为下一次归寂潮储备力量。

## 角色与星域的对应（NovaClash 设定集）

以下角色名、标签名与游戏内及规则说明一致。

- **均衡战士A / 均衡战士B / 均衡战士C**：来自平衡星域的普通星灵，由稳定的三棱水晶构成，无标签，是最可靠的基石。
- **自爆步兵**：诞生于碎晶星域，体内封存不稳定的爆炸水晶；出手后自身必死。若目标最大血量≤26 则直接击杀，否则按攻击力造成伤害；若双方均为自爆步兵则同归于尽。规则上仅能放在队伍首位；不可被死灵法师复活。
- **诅咒巫师**：来自幽暗星域，战斗开始时用暗影水晶剥夺对方一名有标签角色的全部标签（不剥夺对方诅咒巫师，也不以自爆步兵为目标）。
- **死灵法师**：重生星域的守护者，己方角色被击杀时可消耗次数将其以 30% 最大血量（向下取整）复活（自爆步兵除外），有次数限制，死灵法师死亡后失效。
- **铁甲卫士**：重装星域的战士，由致密晶钢构成；受到 ≥6 点伤害时按 60% 结算。
- **护盾部署者**：守护星域的支援者，为队友分担 50% 伤害至自身，有次数限制；自爆伤害不参与分担。
- **风行射手**：迅流星域的猎手，箭矢由高速旋转的水晶棱锥组成；对脆皮（最大血量≤24）命中后可对另一名敌人追加减半伤害。
- **狂战士**：怒火星域的狂徒，血量≤14 时伤害 ×1.4。
- **毒药投手**：蚀骨星域的异类，投掷的晶石使目标中毒，在后续轮次持续造成伤害，可叠加（有上限）。
- **重炮统领**：毁灭星域的战争机器，一次攻击对对方所有存活单位造成等量伤害（群伤）。

### 进阶角色（13-20 号）

- **均衡战士 D / 均衡战士 E**：来自平衡星域的新星灵。D 由尖锐的菱面水晶构成，攻击倾向更明显；E 则由更厚实的立方水晶组成，防御能力更强。
- **暗影猎手**：来自迅流星域的暗影刺客，由半透明的黑曜水晶与银色晶线构成。箭矢能在穿透一个敌人后继续追击另一个目标，对脆皮单位尤为致命。
- **血怒斗士**：怒火星域的精英战士，由赤红裂纹水晶构成。当血量降至 14 以下时，体内水晶会剧烈震动并发出炽热红光，伤害提升至 1.4 倍。
- **晶壁守卫**：重装星域的精英防御者，由多层叠加的半透明晶壁构成。当受到≥6 点伤害时，晶壁会自动折叠压缩，使伤害降低 40%。
- **噬魂者**：来自幽暗星域的异类，由深紫色吞噬水晶构成。每次攻击命中时，会将造成伤害的 25% 转化为自身血量（单次上限 5），但无法通过自爆获取生命。
- **荆棘守卫**：守护星域的特殊防御者，体表布满尖刺状的反向水晶棘刺。当受到攻击时，会将受到伤害的 40% 反弹给攻击者，反弹伤害不触发护盾或重装效果。
- **处决者**：毁灭星域的终结猎手，由锋利的刀锋状水晶构成。当目标当前血量低于最大血量 25% 时，处决者的攻击会触发斩杀效果，伤害提升至 1.5 倍。

每一场战斗，都是星光的交响曲。当你的战士在星核竞技场中释放技能时，整个星陨界的低多边形地貌都会随之共鸣，光粒子如流星般划过战场。失败的一方也不会真正消亡，只是化作散落的水晶碎片，等待下一次归寂潮时被重新唤醒。
